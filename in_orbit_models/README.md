# in_orbit_models
Important mathematical models for ADCS.

#### Folder structure
1. [igrf](https://github.com/risherlock/ADCS/tree/main/in_orbit_models/igrf) - The International Geomagnetic Reference Field (IGRF), a standard mathematical model of the Earth's magnetic field.
2. [sun](https://github.com/risherlock/ADCS/tree/main/in_orbit_models/sun) - Mathematical model to estimate the direction vector of Sun.
3. [time](https://github.com/risherlock/ADCS/tree/main/in_orbit_models/time) - Conversion between some standard times.
